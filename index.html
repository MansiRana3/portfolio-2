What I've built

01 / GenAI

Gemini Persona Roleplay Engine

A CLI chatbot where users chat with fictional characters powered by Google Gemini AI. Built to understand LLMs from first principles not just use them.

  • Implemented Gemini API two ways — raw REST (manual HTTP + JSON) and official google-genai SDK — to understand both abstraction levels
  • Built multi-turn conversation memory from scratch using message history, the same stateless LLM workaround used by ChatGPT
  • Applied 5 prompting techniques (zero-shot, few-shot, system, role, chain-of-thought) each implemented separately to observe distinct behaviour
  • Strict Separation of Concerns, API modules, and prompt logic are fully independent

02 / Deep Learning

GAN Based Synthetic Audio Detection

A deep learning pipeline that classifies real vs AI generated deepfake audio in real time, deployed as a live prediction dashboard.
